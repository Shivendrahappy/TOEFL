# TOEFL Reading Notes — EXCEPT Questions

## Main Idea

In EXCEPT questions:

Three options are TRUE or mentioned in the passage.

One option is NOT mentioned or is FALSE.

That FALSE/NOT MENTIONED option becomes the correct answer.

---

# Strategy

## Step 1 — Notice Trigger Word

Carefully look for words like:

* EXCEPT
* NOT
* LEAST

These completely change the logic of the question.

---

## Step 2 — Identify Keywords

Underline important words in each option.

Example from screenshot:

A → dust, sunlight
B → cold, dark
C → elements, crust
D → nitric acid

---

## Step 3 — Scan Passage

Do NOT read every line deeply again.

Quickly scan for:

* same words
* synonyms
* paraphrases

---

# Example Analysis

## Option A

“dust blocked sunlight”

Passage says:

* dust lifted into atmosphere
* sunlight blocked

So A is TRUE.

---

## Option B

“cold and dark”

Passage says:

* Earth entered cold and darkness

So B is TRUE.

---

## Option D

“nitric acid produced”

Passage directly mentions:

* vast quantities of nitric acid

So D is TRUE.

---

## Option C

“new elements formed in Earth’s crust”

Passage NEVER says this.

Therefore:
C is the correct answer because it is the FALSE/NOT MENTIONED statement.

---

# Important TOEFL Insight

TOEFL usually uses:

* synonyms
* paraphrasing
* reordered sentences

NOT exact copy-paste wording.

So focus on:

* meaning
* paraphrases
* keyword matching

instead of searching exact same sentence.
